# RustMaxx Web

This folder is the **migration target** for the current Next.js app (today at repo root).

Until **Phase 3** of the migration plan is completed, the live web app remains at the repo root (`app/`, `components/`, `lib/`, etc.). After migration, this directory will contain:

- `app/` – Next.js App Router
- `components/`
- `lib/`
- `public/`
- `next.config.js`, `tsconfig.json`, etc.

See [docs/architecture/MIGRATION_PLAN.md](../../docs/architecture/MIGRATION_PLAN.md).
